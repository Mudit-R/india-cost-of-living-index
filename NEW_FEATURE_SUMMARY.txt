NEW FEATURE: VALUE-FOR-MONEY CITY RECOMMENDER
================================================================================

WHAT WE BUILT:
--------------
A sophisticated web application that compares Cost of Living against Quality 
of Life to find cities offering the best value based on user preferences.

THE CONCEPT:
------------
Value Score = Quality of Life Index / Cost of Living Index

- Value > 1.0 = Quality exceeds cost (Good value!)
- Value = 1.0 = Quality matches cost (Fair value)
- Value < 1.0 = Cost exceeds quality (Poor value)

EXAMPLE:
--------
City         Cost    Quality   Value    Verdict
Mumbai       162.79  180       1.11     Good value despite high cost
Bengaluru    114.76  150       1.31     Excellent value!
Delhi        100.00  100       1.00     Baseline
Solapur      68.20   50        0.73     Cheap but lower quality

HOW IT WORKS:
-------------
1. User provides Quality of Life ratings for cities (Delhi = 100 baseline)
2. System uses actual Cost Index data (from our analysis)
3. Calculates Value Score = Quality / Cost
4. Ranks cities by value score
5. Shows surplus/deficit (Quality - Cost)

THREE INPUT METHODS:
--------------------
1. Upload CSV File
   - Download template with all 50 cities
   - Edit quality ratings
   - Upload for analysis

2. Quick Rating Tool
   - Group cities into tiers (Premium, Above Avg, Avg, Below Avg)
   - Assign rating to each tier
   - Select cities for each tier

3. Manual Entry
   - Rate each city individually
   - Full control over every rating

OUTPUT METRICS:
---------------
For each city, the tool shows:
- Value Score (Quality/Cost ratio)
- Quality Index (user's rating)
- Cost Index (actual data)
- Surplus/Deficit (Quality - Cost)
- Value Category (Excellent/Good/Fair/Poor)

VALUE CATEGORIES:
-----------------
Excellent Value: ≥1.20 (Quality significantly exceeds cost)
Good Value:      1.05-1.19 (Quality exceeds cost)
Fair Value:      0.95-1.04 (Quality matches cost)
Poor Value:      <0.95 (Cost exceeds quality)

USE CASES:
----------
1. Job Relocation: Compare offers considering both salary and quality
2. Retirement Planning: Find affordable cities with good healthcare
3. Startup Location: Balance talent, infrastructure, and costs
4. Student Selection: Find cities with good education at reasonable cost

EXAMPLE SCENARIO:
-----------------
Job Offers:
- Mumbai: ₹20L salary, Cost Index = 162.79
- Pune: ₹16L salary, Cost Index = 99.61

If you rate:

- Pune Quality = 120

Value Scores:
- Mumbai: 140/162.79 = 0.86 (Poor value)
- Pune: 120/99.61 = 1.20 (Excellent value)

Result: Pune offers better value despite lower salary!
